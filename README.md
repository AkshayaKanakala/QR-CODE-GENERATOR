# QR Code Scanner & Generator 🚀

A modern web application to **Generate** and **Scan QR Codes** instantly using JavaScript.



## ✨ Features

✅ Generate QR Codes from text or URLs
✅ Scan QR Codes using camera
✅ Scan QR Codes from uploaded images
✅ Responsive modern UI
✅ Fast and lightweight
✅ Mobile-friendly design

---

## 🖼️ Preview

### QR Generator

Generate custom QR codes instantly.

### QR Scanner

Scan QR codes directly using your webcam.

### Image Scanner

Upload QR images and decode them easily.



## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* QRCode.js
* html5-qrcode
